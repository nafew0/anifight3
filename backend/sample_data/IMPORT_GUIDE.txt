STEP 2: Import Anime (IMPORT THIS FIRST!)
────────────────────────────────────────────────────────────────
1. Click "Anime" in the sidebar
2. Click "IMPORT" button (top right)
3. Choose file: anime_sample.csv
4. Click "Submit"
5. Review preview (should show 10 anime with anime_power_scale)
6. Click "Confirm import"
7. ✓ Success! You should see 10 anime imported with their power scales


STEP 3: Import Characters
────────────────────────────────────────────────────────────────
1. Click "Characters" in the sidebar
2. Click "IMPORT" button (top right)
3. Choose file: characters_sample.csv
4. Click "Submit"
5. Review preview (should show ~40 characters)
6. Click "Confirm import"
7. ✓ Success! You should see all characters imported


STEP 4: Verify Data
────────────────────────────────────────────────────────────────
✓ Anime: Should have 10 entries
✓ Characters: Should have ~40 entries
✓ Game templates: Should have 1 entry ("Standard 6v6", Published)


FILE LOCATIONS (from backend/ directory)
────────────────────────────────────────────────────────────────
Anime CSV:      sample_data/anime_sample.csv
Characters CSV: sample_data/characters_sample.csv


TROUBLESHOOTING
────────────────────────────────────────────────────────────────
❌ If character import fails:
   → Make sure you imported anime FIRST!
   → Anime names in character CSV must match exactly

❌ If validation errors occur:
   → Check character_power is between 1.00-100.00
   → Ensure required "name" field is not empty

❌ If nothing shows in import preview:
   → Check CSV file format (comma-separated)
   → Ensure first row has column headers
